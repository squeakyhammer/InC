# 1. Gate Valve

## 1.1 장점
- 전체 개폐용으로 사용
- 양면 실링이 가능하며 방향성이 없음
- 높은 유량 특성을 가지며, 유체가 밸브 내부를 통과할 때 단면 변화가 없어 압력 손실이 적음
- 제작이 용이
- 다양한 유체 사용 가능

## 1.2 단점
- on/off 용으로만 사용 가능하며, 조절용 사용 불가
- 중간개폐 상태로 사용 시 유속 변화에 따른 캐비테이션 발생 가능(Q=VA -> A가 작아질 때 V상승, 베르누이법칙에 따른 압력 감소)
- 밸브 높이가 높고, 리프트가 크므로 개폐 시간이 오래 걸림
